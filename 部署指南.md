# 🚀 免费部署指南 - Streamlit Cloud

## 部署后效果
- 获得一个公网网址，如：`https://你的应用名.streamlit.app`
- 老师们手机浏览器打开即可使用
- 可以添加到手机桌面，像APP一样

---

## 第一步：注册 GitHub 账号

1. 访问 https://github.com
2. 点击右上角 **Sign up** 注册
3. 填写用户名、邮箱、密码
4. 验证邮箱

---

## 第二步：创建 GitHub 仓库

1. 登录后点击右上角 **+** → **New repository**
2. Repository name 填：`auto-schedule-app`
3. 选择 **Private**（私有）或 **Public**（公开）
4. 点击 **Create repository**

---

## 第三步：上传代码到 GitHub

### 方法A：使用 GitHub Desktop（推荐新手）

1. 下载安装 GitHub Desktop：https://desktop.github.com
2. 登录你的 GitHub 账号
3. File → Add local folder → 选择 `自动调课App` 文件夹
4. Publish repository → 选择你刚创建的仓库 → Publish

### 方法B：网页上传

1. 在仓库页面点击 **uploading an existing file**
2. 把 `自动调课App` 文件夹里所有文件拖进去
3. 点击 **Commit changes**

---

## 第四步：注册 Streamlit Cloud

1. 访问 https://streamlit.io/cloud
2. 点击 **Start for free**
3. 选择 **Sign up with GitHub**
4. 授权 Streamlit 访问你的 GitHub

---

## 第五步：部署应用

1. 登录 Streamlit Cloud 后，点击 **New app**
2. 选择：
   - Repository: `你的用户名/auto-schedule-app`
   - Branch: `main`
   - Main file path: `app.py`
3. 点击 **Deploy!**

等待 2-3 分钟，部署完成后会显示：
```
Your app is live at: https://你的应用名.streamlit.app
```

---

## 第六步：分享给老师们

把网址发给老师们：
```
https://你的应用名.streamlit.app
```

他们：
1. 手机浏览器打开
2. 登录（admin/admin123 或 jiaowu/jiaowu123）
3. 首次使用点击「数据初始化」

---

## ⚠️ 重要：数据持久化

Streamlit Cloud 免费版**每次重启会重置数据库**。

### 解决方案：定期备份数据

我已添加数据导出功能：
- 在「个人中心」可以导出调课记录
- 建议每周导出一次

### 更好的方案（如果需要长期稳定）

1. **免费云数据库**：
   - Supabase (PostgreSQL) - 免费额度很大
   - PlanetScale (MySQL) - 免费额度

2. 我可以帮你改造数据库连接，对接云数据库

## 常见问题

### Q: 部署失败怎么办？
A: 查看 Streamlit Cloud 的日志，通常会提示缺少什么依赖

### Q: 网址打不开？
A: 免费版启动较慢（约30秒），请耐心等待

### Q: 数据丢了？
A: 免费版数据库会重置，建议定期导出，或升级到云数据库
