# 📚 中学自动调课系统 v2.0

基于Python Streamlit的智能调课Web应用，专为中学七至九年级设计。

## 🎯 核心功能

### 1. 智能顶课
- **正课（第1-9节）**：本班任课老师轮流顶替，跳过有课教师
- **晚自习（第10-11节）**：按全校轮值表顺序顺延指派

### 2. 角色管理
- **管理员** (admin/admin123)：维护课表数据、调整晚自习轮值顺序
- **教务** (jiaowu/jiaowu123)：发起顶课、查看调课记录

### 3. 5个功能页面
| 页面 | 功能 |
|------|------|
| 🏠 首页 | 欢迎页、统计概览、快捷入口 |
| 📝 发起顶课 | 选择请假教师→勾选节次→一键生成方案 |
| 📅 课表查询 | 按班级或教师查看周课表 |
| 🌙 晚自习轮值 | 查看/调整轮值顺序 |
| 👤 个人中心 | 我的课表、顶课记录、调课历史 |

## 🚀 快速启动

### Linux/Mac
```bash
cd 自动调课App
pip install -r requirements.txt
python3 schedule_parser.py  # 首次需导入数据
./start.sh
# 或直接: python3 -m streamlit run app.py
```

### Windows
```cmd
cd 自动调课App
pip install -r requirements.txt
python schedule_parser.py  # 首次需导入数据
start.bat
```

### 访问
打开浏览器访问: **http://localhost:8501**

## 🔧 调课规则详解

### 正课顶替规则
1. 找出请假教师的所有正课
2. 对每节课，找出该班级该时段空闲的任课老师
3. 按顺序安排顶替（优先本班任课老师）
4. 已安排的教师不能再顶替其他课

### 晚自习顶替规则
1. 维护全校教师的晚自习轮值表
2. 按顺序循环指派值班教师
3. 自动跳过已被安排的教师
