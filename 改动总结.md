# 自动调课App 改进需求完成总结

## 改动概述

本次更新完成了5个改进需求，涉及3个核心文件的修改。

---

## 1. database.py 改动

### 1.1 数据结构变更
- **substitutions表**：新增 `leave_reason` 字段（请假事宜）
- **新增表**：添加 `evening_assigned` 表用于跟踪晚自习分配记录

### 1.2 新增数据库迁移代码
- 自动检测并添加 `leave_reason` 字段（向后兼容）
- 确保 `evening_assigned` 表存在

### 1.3 新增教师管理方法
| 方法名 | 功能 |
|--------|------|
| `update_teacher_name(teacher_id, new_name)` | 修改教师姓名 |
| `add_teacher(name)` | 添加新教师 |
| `delete_teacher(teacher_id)` | 删除教师（软删除） |
| `reactivate_teacher(teacher_id)` | 重新激活教师 |
| `get_all_teachers_including_inactive()` | 获取所有教师（含已删除） |

### 1.4 新增课表管理方法
| 方法名 | 功能 |
|--------|------|
| `update_schedule_teacher(item_id, new_teacher_id)` | 修改课表任课教师 |
| `get_schedule_item(item_id)` | 获取单条课表记录 |
| `delete_schedule_item(item_id)` | 删除课表记录 |
| `add_schedule_item(...)` | 添加课表记录 |

### 1.5 新增个人中心相关方法
| 方法名 | 功能 |
|--------|------|
| `get_my_leave_records(teacher_id)` | 获取请假记录 |
| `get_teacher_substitutions_filtered(...)` | 获取顶课记录（支持时间筛选） |
| `get_teacher_leave_records_filtered(...)` | 获取请假记录（支持时间筛选） |

### 1.6 新增晚自习轮值方法
| 方法名 | 功能 |
|--------|------|
| `get_evening_assigned_teachers()` | 获取已安排轮值的教师 |
| `get_evening_unassigned_teachers()` | 获取未安排轮值的教师 |
| `add_teacher_to_evening_rotation(teacher_id)` | 添加教师到轮值表 |
| `remove_teacher_from_evening_rotation(teacher_id)` | 从轮值表移除教师 |
| `reorder_evening_rotation(ordered_ids)` | 重新排序轮值表 |

---

## 2. app.py 改动

### 2.1 侧边栏导航
- 新增「⚙️ 数据管理」入口

### 2.2 发起顶课页面（需求2、3）
**新增功能：**
- 请假事宜输入框（步骤1）
- 调课方案中显示请假事宜
- 「下一个」按钮：自动切换到下一个可用教师
- 冲突检测：显示当前选择教师是否有冲突
- 导出CSV新增「请假事宜」列

### 2.3 晚自习轮值页面（需求4）
**改造为双列布局：**
- 左列：✅ 已安排轮值的教师（按姓名排序）
- 右列：⏳ 未安排轮值的教师（按姓名排序）
- 每位教师旁边有「➕ 添加」或「❌ 移除」按钮
- 保留管理员手动调整顺序功能

### 2.4 个人中心页面（需求5）
**重构为Tab布局：**
- Tab1：📌 顶课记录（帮别人顶的课）
  - 显示：时间、教师、班级、请假事宜、分配方式
  - 支持按时间筛选
  - 支持导出CSV
- Tab2：🏖️ 请假记录（自己请假被顶的课）
  - 显示：时间、顶课教师、班级、请假事宜
  - 支持按时间筛选
  - 支持导出CSV
- 保留「所有调课记录」功能（管理员/教务可见）

### 2.5 新增数据管理页面（需求1）
**三个Tab页：**
1. **👨‍🏫 教师管理**
   - 添加新教师
   - 修改教师姓名
   - 删除/恢复教师
   - 区分活跃/已删除教师

2. **📅 课表管理**
   - 选择班级查看课表
   - 修改任意课表的任课教师
   - 实时保存修改

3. **🏫 班级管理**
   - 查看所有班级
   - 显示每个班级的课程数量

4. **⚙️ 数据初始化**
   - 从Excel导入课表
   - 清空所有数据
